Detailed Description

Namespace for free functions related to Logging.

Spdlog provides a singleton logger which can be accessed anywhere, this namespace groups functions which configure spdlog's default logger.

void Controller::Logger::attachConsole ( )

Attaches the console to the logger.

The console is provided as a sink for the default logger to write logs to. Logs written to the console do not include as many details as those written to files.
